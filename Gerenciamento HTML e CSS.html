Gerenciamento de Configuração de Software com HTML e CSS

Controle de Versão: Controle de versão é um programa que guarda alterações em versões, designers gráficos que desejam manter todas as versões de um arquivo de imagem devem usar o sistema de controle de versão (VCS), já que ele permite que você abra o estado anterior de arquivos, veja quem fez as mudanças, e caso perca os arquivos, possa usar uma versão anterior para corrigi-los facilmente.

Controle de mudanças: O controle de mudanças é o que permite a visualização de cada mudança feita no programa e quem a fez, como dito anteriormente, ele é encontrado no controle de versões.

Integração Contínua: Na integração contínua os programadores devem com frequência atualizarem o código e sem parar por muito tempo, assim impedindo problemas que seriam causados caso todos os desenvolvedores decidissem enviar seus arquivos no fim da produção de um projeto.
